Transitions

Transitions are when the browser animates properties from one state to another. These are best suited to things like hover effects or adding / removing classes to elements to change their state. An example might be if an item goes from "normal" to "urgent" on a list and it's background color changes - a transition can help make the change more noticeable.

transition: all .5s ease-out;

Animations

Keyframes and animations are more advanced, and more powerful than transitions. They're when the browser animates through a series of states, called "keyframes". Animations can be played forwards, backwards and more. The playing state of an animation can be controlled by JavaScript.

animation: animation-name .5s ease-out forwards;

@keyframes animation-name {
  from { opacity: 0; }
  to { opacity: 1; }
}
